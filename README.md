# whoami ?

## This website captures some of my interests and my identity

> "Feet on ground, eyes on stars."

A minimal, warm, and personal digital space. Built simply with vanilla HTML, CSS, and JavaScript.

## features
- **minimal design**: a warm cream-and-accent aesthetic inspired by nature and simple books.
- **programming to poetry**: sections covering systems programming, poetry (कवितेचे पान), thoughts, books, and dance.
- **dynamic integration**: auto-loaded github pinned projects and book covers.
- **pure and fast**: no frameworks, no build steps, optimized fonts, and clean vector iconography.
